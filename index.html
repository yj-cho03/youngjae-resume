My Work

내가 맡은 일

촬영 이후 데이터를 바로 분석 가능한 형태로 연결

라즈베리파이 카메라로 이미지를 촬영하는 단계에서 끝내지 않고, 이후 좌표 추출과 JSON 생성까지 이어질 수 있도록 전체 흐름을 고려해 작업했습니다.

관절 좌표를 실제 활용 가능한 신체 데이터로 변환

MediaPipe Pose로 추출한 raw landmark 데이터를 그대로 두지 않고, 어깨 너비, 허리 너비, 상체 길이, 하체 길이와 같은 길이 값과 비율 값으로 가공했습니다.

비율 중심 분석 구조 설계

촬영 거리나 자세 차이에 따라 절대 길이 값이 달라질 수 있다는 점을 고려해, 결과 해석에는 상체·하체 비율과 어깨·허리 비율 같은 상대값이 활용되도록 정리했습니다.

Process

처리 흐름

01 이미지 촬영
Raspberry Pi Camera Module을 이용해 사용자 전신 이미지를 정면과 측면 기준으로 수집합니다.
02 좌표 추출
OpenCV와 MediaPipe Pose를 활용해 33개 신체 관절 좌표를 추출합니다.
03 JSON 생성
추출된 좌표 데이터를 front.json, side.json 구조로 저장하고 후속 처리에 활용합니다.
04 길이/비율 계산
어깨, 허리, 상체, 하체 관련 길이 값을 계산하고 비율 정보를 body_result.json으로 정리합니다.
05 서버 활용
Python 기반 Fast API 서버와 REST API(JSON) 방식으로 연결해 분석 결과가 시스템에서 활용될 수 있도록 구성합니다.
