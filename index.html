프로젝트 개요

사용자의 전신 이미지를 촬영하고 MediaPipe Pose를 활용하여 신체 관절 좌표를 추출한 뒤, 길이 및 비율 데이터를 계산하여 체형 분석에 활용할 수 있는 JSON 데이터를 생성하는 시스템입니다.

담당 역할

  • Raspberry Pi Camera Module 기반 전신 이미지 촬영 환경 구성
  • MediaPipe Pose 기반 33개 신체 관절 좌표 추출
  • 어깨 너비, 허리 너비, 상체 길이, 하체 길이 계산 로직 구현
  • 상체/하체 비율, 어깨/허리 비율 계산 구조 설계
  • front.json, side.json, body_result.json 생성 및 구조화

개발 환경

Python Fast API Flutter Web REST API (JSON) Raspberry Pi OpenCV MediaPipe Pose MySQL

결과

촬영 → 좌표 추출 → 길이 및 비율 계산 → JSON 생성 → 서버 활용으로 이어지는 End-to-End 체형 분석 파이프라인을 구현했습니다.
